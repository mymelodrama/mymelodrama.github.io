最长回文子串

​ 给定一个字符串 s，找到 s 中最长的回文子串。

算法一：寻找最长公共子串

​ 反转字符串 S*，使之变成S′。找到 S 和 S′ 之间最长的公共子串，这也必然是最长的回文子串。

似乎方法可行，例如S=“1221abc” S′=“cba1221” S∩S′=“1221” 最长回文子串为“1221”。

不过当这个例子： S=“abacdfgdcaba”, S′=“abacdgfdcaba” S∩S′=“abacd”，显然，这甚至不是回文串。

所以在搜寻基础上加入判断是否为回文串就是必须的。

贴上c++

算法二：暴力查找

​ 穷举出所有的子字符串，判断是否为回文串

算法三：扩展中心

​ 由于回文串是中心对称 ，每次循环选择一个中心，进行左右扩展，判断左右字符是否相等即可。由于存在奇数的字符串和偶数的字符串，所以我们需要从一个字符开始扩展，或者从两个字符之间开始扩展，所以总共有 n+n-1 个中心。（奇数有n种中心，偶数有n-1种中心）


只出现一次的数

​ 给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

如[2,2,1] 输出1

[4,1,2,1,2] 输出4

算法一：穷举法

时间复杂度o(n^2)

过于复杂，不建议使用。

算法二：简单的数学异或运算

1.a^b^c==a^c^b(交换律)

2.n(任意数)^0==n

3.n^n==0。

由上面三条定律只需将0分别和数组中的所有元素异或即可得到只出现一次的数字

分割链表

给定一个链表和一个特定值 x，对链表进行分隔，使得所有小于 x 的节点都在大于或等于 x 的节点之前。你应当保留两个分区中每个节点的初始相对位置。

示例:

输入: head = 1->4->3->2->5->2, x = 3
输出: 1->2->2->4->3->5

本题要求改变链表结构，使得值小于 x的元素，位于值大于等于x元素的前面。这实质上意味着在改变后的链表中有某个点，在该点之前的元素全部小于x ，该点之后的元素全部 大于等于x。

双指针法：
用两个指针find_big 和 find_small 来追踪上述的两个链表。两个指针可以用于分别创建两个链表，然后将这两个链表连接即可获得所需的链表。

删除链表的倒数第N个节点

给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

运用双指针法，一快指针，一慢指针，一开始，快指针从列表的开头向前移动 n+1步，而慢指针将从列表的开头出发。现在，这两个指针被 n个结点分开。通过同时移动两个指针向前来保持这个恒定的间隔，直到快指针到达最后一个结点。此时慢指针将指向从最后一个结点数起的第 n +1个结点。删去慢指针的后一节点即可。
